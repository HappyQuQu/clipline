# Clipline UI Flow

## 1. 文档目的

本文档定义 Clipline MVP 的页面结构、核心交互流程、空状态和错误状态，供产品、设计、前端、后端联调使用。

MVP 以桌面端体验为主，移动端保证基础可用。

## 2. 信息架构

MVP 包含三个主页面：

```text
回看页
设置页
导出任务页
```

全局导航：

- 左侧窄导航栏承载主要模块入口。
- 左侧控制栏提供源选择、日期选择、新建源、当前源状态和扫描操作。
- 主工作区以视频播放器为核心。
- 播放器下方提供导出区间和录像片段卡片墙。
- MVP 界面不展示应用日志；日志只保留为后端诊断接口。

## 3. 回看页

回看页是用户进入应用后的主页面。

### 3.1 页面目标

- 按源和日期查看录像片段。
- 通过时间线快速定位真实时间。
- 播放当前片段。
- 连续播放后续片段。
- 选择时间范围并导出。

### 3.2 页面布局

```text
+------+----------------+----------------------------------------+
| 导航 | 控制栏          | 主工作区                                |
|      | - 源选择        | - 大尺寸视频播放器                      |
|      | - 日期          | - 左侧时间轨                            |
|      | - 新建源        | - 悬浮播放/导出/扫描工具                |
|      | - 当前源状态    | - 导出区间                              |
|      | - 最近扫描      | - 录像片段卡片墙                        |
+------+----------------+----------------------------------------+
```

视觉参考 UniFi Protect：浅灰背景、白色工作区、细边框、紧凑间距、蓝色操作态、视频内容优先。

### 3.3 首次进入

如果还没有任何源：

- 显示无源空状态。
- 主操作按钮为“新建源”。
- 点击后进入设置页的新建源弹窗或抽屉。

文案：

```text
还没有配置录像源
请先新建一个源，并选择容器内的视频目录。
```

### 3.4 已有源但当天无视频

条件：

- 已选择源。
- 已选择日期。
- 当天没有扫描到视频片段。

展示：

- 播放器区域显示空状态。
- 时间线显示完整空白区间。
- 片段列表为空。

文案：

```text
这一天没有找到录像片段
```

操作：

- 切换日期。
- 切换源。
- 手动重新扫描。

## 4. 设置页

设置页用于管理源、查看系统状态和清理缓存。

### 4.1 页面模块

- 源管理。
- 扫描状态。
- 系统状态。
- P1 缩略图缓存。
- 导出缓存。

## 5. 新建源流程

### 5.1 入口

用户可以从两个地方新建源：

- 回看页无源空状态。
- 设置页源管理模块。

### 5.2 表单字段

必填：

- 源名称。
- 视频目录。

系统自动处理：

- 源 ID。
- 应用时区。
- 自动时间识别规则。

### 5.3 表单布局

```text
新建源

源名称
[ 门口摄像头                 ]

视频目录
[ /recordings/front-door      ] [选择目录]

目录树
/recordings
  front-door/
  dashcam/
  living-room/

[取消] [保存并扫描]
```

### 5.4 目录选择器

目录选择器只展示允许访问的录像根目录。

默认根目录：

```text
/recordings
```

目录选择规则：

- 可以展开子目录。
- 可以选择任意可读目录。
- 不展示隐藏目录。
- 不展示 `/app/data` 等应用数据目录。
- 目录不可读时置灰，并显示不可读状态。
- 支持手动输入容器内路径，作为高级兜底方式。

### 5.5 保存校验

保存前校验：

- 源名称不能为空。
- 视频目录不能为空。
- 视频目录必须存在。
- 视频目录必须可读。
- 视频目录必须位于允许的录像根目录内。
- 视频目录不能与已有源完全相同。

保存成功：

- 创建源。
- 自动触发首次扫描。
- 返回设置页源列表。
- 显示扫描状态。

保存失败：

- 保留用户输入。
- 在对应字段下显示错误原因。

## 6. 源管理流程

### 6.1 源列表

每个源展示：

- 源名称。
- 视频目录。
- 启用状态。
- 最近一次扫描时间。
- 扫描状态。
- 已索引片段数。
- 错误数。

可操作：

- 编辑。
- 启用或停用。
- 重新扫描。

### 6.2 编辑源

可编辑：

- 源名称。
- 视频目录。

编辑视频目录后：

- 系统提示用户需要重新扫描。
- 保存后触发扫描。

### 6.3 停用源

停用后：

- 回看页不默认展示该源。
- 不再自动扫描该源。
- 历史索引保留。

## 7. 扫描流程

### 7.1 自动扫描

触发时机：

- 新建源成功后。
- 按配置间隔周期扫描。
- 应用启动后可触发一次扫描。

### 7.2 手动扫描

入口：

- 设置页源列表。
- 回看页空状态。

扫描状态：

- queued。
- running。
- completed。
- failed。

展示信息：

- 已扫描文件数。
- 已入库文件数。
- 失败文件数。
- 当前进度。
- 最近错误。

## 8. 时间线交互

### 8.1 时间线状态

时间线覆盖当天 00:00 到 24:00。

状态：

- 有录像。
- 无录像。
- 当前播放时间。
- 选中导出区间。
- 加载中。
- 片段损坏或不可播放。

### 8.2 点击时间线

用户点击有录像区域：

- 系统解析真实时间到具体片段。
- 播放器加载该片段。
- 设置片段内偏移。
- 时间线指针移动到点击时间。

用户点击无录像区域：

- 显示最近可播放时间提示。
- 提供跳转到上一段或下一段按钮。

### 8.3 拖动时间线

拖动过程中：

- 显示当前真实时间。
- P1 如果有缩略图，显示预览图。

松开后：

- 如果目标时间有录像，跳转播放。
- 如果目标时间无录像，提示最近可播放片段。

## 9. 播放流程

### 9.1 播放片段

用户点击片段：

- 播放器加载片段 stream 地址。
- 播放器跳到片段内偏移。
- 片段卡片高亮。
- 时间线指针更新。

### 9.2 连续播放

当前片段播放结束：

- 查询下一个时间顺序片段。
- 如果时间连续，自动切换播放。
- 如果存在断档，显示断档时长后切换到下一片段。
- 如果没有下一片段，停留在结束状态。

### 9.3 不可播放文件

条件：

- 浏览器不支持编码。
- 文件损坏。
- Range 请求失败。

展示文案：

```text
该片段暂时无法在线播放
可以尝试导出，或等待后续转码预览能力。
```

操作：

- 导出该时间段。
- 跳到下一片段。

## 10. 导出流程

### 10.1 进入导出模式

入口：

- 回看页“导出”按钮。
- 片段列表中的导出操作。

进入后：

- 时间线进入区间选择状态。
- 播放控制保持可用。

### 10.2 选择区间

方式：

- 在时间线上拖拽选择开始和结束时间。
- 使用时间输入框微调开始和结束时间。

展示：

- 开始时间。
- 结束时间。
- 预计时长。
- 涉及片段数。
- 是否包含断档。

如果包含断档，界面必须提示：

- 导出时会跳过断档。
- 输出文件只包含实际存在的录像内容。
- 输出视频时长可能短于选择区间。

### 10.3 选择导出模式

模式：

- 快速导出。
- 精准导出。

默认：

- 快速导出。

模式说明：

- 快速导出速度快，但裁剪点可能不完全精确。
- 精准导出更准确，兼容性更好，但 CPU 占用更高。

### 10.4 提交导出

提交后：

- 创建导出任务。
- 跳转或提示进入导出任务页。
- 回看页可继续使用。

## 11. 导出任务页

### 11.1 页面目标

- 查看导出任务状态。
- 下载已完成文件。
- 查看失败原因。
- 清理过期导出。

### 11.2 任务列表字段

- 源名称。
- 时间范围。
- 模式。
- 状态。
- 进度。
- 输出文件大小。
- 创建时间。
- 过期时间。

### 11.3 任务状态

- queued：排队中。
- running：导出中。
- completed：已完成。
- failed：失败。
- expired：已过期。

### 11.4 任务操作

可操作：

- 下载。
- 重新导出，复用 `POST /api/exports`。P1 能力。
- 删除导出文件。P1 能力。

## 12. 空状态与错误状态

### 12.1 无源

```text
还没有配置录像源
请先新建一个源，并选择容器内的视频目录。
```

主操作：

- 新建源。

### 12.2 源目录不可读

```text
无法读取该视频目录
请检查 Docker volume 是否已正确挂载，且路径位于 /recordings 下。
```

主操作：

- 编辑源。
- 重新扫描。

### 12.3 当天无视频

```text
这一天没有找到录像片段
```

主操作：

- 切换日期。
- 重新扫描。

### 12.4 扫描失败

```text
扫描未完成
部分文件无法读取或解析，请查看扫描详情。
```

主操作：

- 查看错误。
- 重新扫描。

### 12.5 导出失败

```text
导出失败
可以尝试改用精准导出，或缩短导出时间范围。
```

主操作：

- 重新导出。
- 改用精准导出。

## 13. 移动端基础适配

MVP 不做完整移动端优化，但应保证可基础使用：

- 顶部源选择和日期选择可用。
- 播放器宽度自适应。
- 时间线支持横向滚动。
- 左侧栏折叠为抽屉。
- 片段列表单列展示。
- 导出区间输入框在窄屏下上下排列。
